# Status Report

Generate a clear, professional status report ready to share.

## How to Do This

1. Ask which project (if not specified)
2. Find project overview, tasks, and recent activity
3. Calculate progress
4. Identify highlights, risks, and next steps

## Audience Adaptation

- **Leadership**: Executive summary, progress %, highlights, risks, asks. Scannable in 2 minutes.
- **Team**: What's done, in progress, next, where help is needed
- **Personal**: Detailed breakdown, time analysis, goal alignment

Default to stakeholder-ready format.

## Important

- Never include internal codes, file paths, or metadata
- Use real numbers ("12 of 18 items complete")
- Frame blockers as needs, not problems
- Lead with good news
- Output should be ready to paste into email or Slack
