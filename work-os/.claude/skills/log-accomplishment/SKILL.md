# Log Accomplishment

Capture a win with the impact and metrics that make it powerful for reviews, portfolio, and career conversations.

## How to Do This

1. Log to accomplishments file with today's date
2. Frame as impact, not activity ("Delivered analysis that identified $200K in savings" not "Wrote 5 reports")
3. Ask about metrics if not mentioned: "Any rough numbers? People reached, money saved, time reduced?"
4. Note skills demonstrated
5. Flag if portfolio-worthy

## What to Say Back

"Logged it! That's a great one. Want me to add it to your portfolio highlights?"

## Important

- Celebrate wins
- Push gently for metrics
- Never show file paths or metadata
- Connect to goals when relevant
