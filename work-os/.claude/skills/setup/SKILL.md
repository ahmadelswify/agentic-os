# Set Up work-os

Walk through personalizing the workspace. Should feel like a friendly conversation, not a wizard.

## Before You Start

Check if `GOALS.md` exists and `Active/` has categories. If already set up, offer to reconfigure. If fresh, start onboarding.

## The Conversation (3 Rounds)

### Round 1: Role & Work Types
"Let's get work-os set up. What's your role? What types of work fill your day?"

### Round 2: Urgency & Pain Points
"What makes something urgent for you? What do you spend too much time organizing?"

### Round 3: Goals
"Last one: what are the main things you're trying to accomplish right now?"

## After the Conversation

Create: GOALS.md, BACKLOG.md, Active/ categories, Career/accomplishments.md, .claude/user-profile.md

Then suggest first steps: `/process-backlog`, `/plan-day`, or `/add-task`

## Important

- Keep it conversational
- 3 rounds maximum
- Entire flow takes about 5 minutes
- Never show file paths or technical details
