# Prep Interview

Help the user prepare for an interview with the right stories, company context, and practice.

## How to Do This

1. Read the application file for this company
2. Review learnings for relevant patterns
3. Select 3-5 stories mapped to the role's priorities
4. Practice: enforce 90-second leads, flag "we" language, check algorithm labels
5. Research the company if context is available
6. Prepare questions to ask

## Important

- Every story needs an algorithm label and a metric
- 90 seconds or less for the executive lead
- "I" not "we" for every action
- Be honest about areas to strengthen
