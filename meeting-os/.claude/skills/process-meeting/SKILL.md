# Process Meeting

The user just finished a meeting. Process their notes or transcript into structured records.

## How to Do This

1. Create a meeting note in Meetings/ with summary, decisions, and action items
2. Extract and save decisions to Decisions/
3. Extract and save action items to Actions/
4. Update People/ profiles
5. Create follow-up reminders
6. Summarize what was captured

## Important

- Keep summaries to 3-5 bullets
- Confirm ambiguous action item ownership
- Never show file paths or metadata
