# Process Inbox

Turn brain dumps into structured tasks linked to life goals.

## How to Do This

1. Read INBOX.md and GOALS.md
2. Classify each item: task, reference, idea, or unclear
3. Create task files in Tasks/ with proper categories and priorities
4. Link each task to a goal
5. Clear processed items
6. Summarize what was created

## Important

- Every task must link to a goal
- Batch questions together
- Never show YAML or priority codes
