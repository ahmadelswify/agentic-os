# Add Task

Capture a task from natural language and organize it behind the scenes.

## How to Do This

1. Parse: task, urgency, due date, category, goal connection
2. Create task file with YAML frontmatter (invisibly)
3. Confirm: "Got it! Added '[task]' to [category], due [date]."

## Important

- FAST: capture and confirm in one exchange
- Never show YAML or file paths
- Mention goal connection if you see one
